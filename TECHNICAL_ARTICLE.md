# 我花1天给Trae做了个三层记忆插件，开源免费

## 🚀 背景故事

作为一名AI开发者，我每天都在使用各种AI助手进行编程和创作。但有个问题一直困扰着我：**AI助手记不住我！** 每次新对话都要重新介绍自己、重新解释项目背景、重新讨论技术方案。

这就像每次见面都要重新认识的朋友，效率实在太低了！

于是，我决定用1天时间，为Trae IDE开发一个**三层智能记忆插件**，让AI助手真正拥有长期记忆能力！

## 🧠 三层记忆架构设计

### L1 - 长期记忆（重要信息）
- **存储内容**：重要决策、关键信息、用户偏好
- **识别方式**：自动检测"记住..."、"我叫..."、"项目重点是..."等关键词
- **保存策略**：永久保存，不会被自动清理

### L2 - 情景记忆（近期对话）
- **存储内容**：近期对话、项目上下文、工作状态
- **处理方式**：自动摘要化存储
- **压缩策略**：定期压缩，保留精华

### L3 - 会话记忆（当前对话）
- **存储内容**：当前对话、临时信息、缓存数据
- **存储方式**：原始对话存储
- **清理策略**：自动清理，释放空间

## 🔧 技术实现

### 智能筛选器
```python
# 自动识别重要信息
def smart_filter(text):
    if "记住" in text or "我叫" in text or "项目重点" in text:
        return "L1"  # 长期记忆
    elif "错误" in text or "功能" in text or "决定" in text:
        return "L2"  # 情景记忆
    else:
        return "L3"  # 会话记忆
```

### 智能检索器
- **话题感知**：检测当前话题关键词，自动检索相关记忆
- **渐进式披露**：先显示摘要，需要时再展开全文
- **记忆预热**：新会话自动加载最相关的5条记忆

### 智能压缩器
- **自动摘要**：每10轮对话生成200字摘要
- **重复合并**：相似内容自动合并去重
- **空间预警**：当存储超过1G时提醒整理

## 🎯 核心功能亮点

### 1. 自动识别重要信息
插件能智能识别用户输入中的重要信息：

- 第1次提到"记忆系统" → L2情景记忆
